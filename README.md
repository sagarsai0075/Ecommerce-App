# Ecommerce-App (ShopEazy)

A full-stack ecommerce application built with Angular (frontend) and Node.js/Express (backend) . This project allows users to browse products, manage carts, place orders, and sellers to manage inventory.

## Features
- User authentication & authorization
- Product listing, search, and filtering
- Shopping cart management
- Order placement and tracking
- Seller dashboard for product management
- Payment integration
- Admin panel for managing products and orders

## Tech Stack
- **Frontend:** Angular
- **Backend:** Node.js, Express
- **Database:** MongoDB

## Getting Started
### Backend
1. Navigate to the Backend folder:
   cd Backend
2. Install dependencies:
   npm install
3. Start the server:
   npm start

### Frontend
1. Navigate to the frontend folder:
   cd frontend
2. Install dependencies:
   npm install
3. Start the Angular app:
   ng serve

## Environment Variables
- Configure environment variables in Backend/.env and frontend/src/environments/environment.ts as needed.
